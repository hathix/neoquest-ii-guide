boss levels

You know how I tell you all the recommended levels you should be to fight each boss during the guide? Well, I've compiled all the stuff from the guide right here, so you can find the recommended levels for Zombom or Hubrid Nox all in the same table. It's all below. Also, if you want more info on any boss, just click on the boss's name. More info will open in a new tab.

How to read the table: First, find the boss's name. Then read across the columns: go over one column if you're on Normal, 2 if Evil, and 3 if Insane. The number you land on is the recommended level. For example, look for Zombom. You should be level 12 if facing him on Normal, 13 if you're on Evil, and 14 if you're on Insane.
